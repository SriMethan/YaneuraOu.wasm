# About this project

KomoringHeights は、df-pnアルゴリズムを用いたやねうら王ベースの詰将棋ソルバーです。
ミクロコスモスをはじめとする長手数の詰将棋を解くことを目的としています。

KomoringHeights 本体は `source/engine/user-engine` 以下に格納されています。
